libpysal.cg.PolygonLocator¶

class libpysal.cg.PolygonLocator(polygons)[source]¶

An abstract representation of a polygon indexing data structure.

Examples

>>> p2 = Polygon([Point((3, 9)), Point((6, 7)), Point((1, 1))])

True
